Entorno y Configuracion de Express.js

Instalar Node.js y npm:

Asegúrate de tener Node.js y npm instalados en tu sistema. Puedes descargarlos e instalarlos desde el sitio web oficial de Node.js.

Crear un nuevo proyecto:

Crea un nuevo directorio para tu proyecto y abre una terminal en ese directorio.

Inicializar el proyecto Node.js:

Ejecuta el siguiente comando para iniciar un nuevo proyecto Node.js. Esto creará un archivo package.json que contendrá la información del proyecto.

$ npm init -y
Instalar Express:

Instala el paquete de Express usando npm:

$ npm install express
Crear el archivo de aplicación Express:

Crea un archivo app.js (o cualquier otro nombre que prefieras) para tu aplicación Express y abre el archivo en tu editor de texto preferido.

Editar el archivo app.js:

Dentro de app.js, configura tu aplicación Express básica. Puedes empezar con un "Hola, mundo" simple:

// app.js
const express = require('express');
const app = express();
const port = 3000;

app.get('/', (req, res) => {
  res.send('¡Hola, mundo!');
});

app.listen(port, () => {
  console.log(`La aplicación está escuchando en http://localhost:${port}`);
});
Ejecutar la aplicación:

Ejecuta tu aplicación con el siguiente comando, donde nombre-del-archivo es el nombre que le hayas asignado al archivo anterior.

$ node nombre-del-archivo.js
Visitar la aplicación:

Abre tu navegador y visita http://localhost:3000. Deberías ver el mensaje "¡Hola, mundo!".
