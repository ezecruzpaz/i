Generar Archivo PDF con Express.js y pdfkit

Para generar un archivo PDF en una aplicación Express.js, puedes utilizar una librería como pdfkit junto con express para manejar las rutas y las respuestas HTTP. A continuación, te proporciono un ejemplo básico de cómo hacerlo:

Instala las dependencias necesarias:
npm install express pdfkit
Crea un archivo app.js o index.js y añade el siguiente código:
const express = require('express');
const PDFDocument = require('pdfkit');

const app = express();
const port = 3000;

app.get('/generate-pdf', (req, res) => {
  // Crea un nuevo documento PDF
  const doc = new PDFDocument();

  // Establece el tipo de contenido del encabezado de la respuesta
  res.setHeader('Content-Type', 'application/pdf');

  // Establece el encabezado para indicar la descarga del archivo PDF
  res.setHeader('Content-Disposition', 'attachment; filename=generated.pdf');

  // Conecta el documento PDF a la respuesta HTTP
  doc.pipe(res);

  // Agrega contenido al PDF
  doc.fontSize(16).text('Hola, este es un PDF generado con Express.js y pdfkit.');

  // Finaliza el documento PDF
  doc.end();
});

// Inicia el servidor Express
app.listen(port, () => {
  console.log(`Servidor Express escuchando en el puerto ${port}`);
});
Ejecuta la aplicación:
node app.js
Accede a http://localhost:3000/generate-pdf en tu navegador. Esto debería iniciar la descarga del archivo PDF generado.

Este es un ejemplo simple, y puedes personalizar el contenido del PDF según tus necesidades. Puedes consultar la documentación de pdfkit (https://pdfkit.org/) para obtener más información sobre cómo agregar texto, imágenes y otros elementos al PDF.
